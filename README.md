# Deep Dive — Endless Descent

This is a browser-based endless underwater descent game. You control a diver sinking into the deep ocean, dodging the weird algae.

## What is this project?
**Deep Dive** is a casual arcade game built with pure HTMLl, CSS, Canvas and JavaScript. The player steers a glowing diver left and right while rock formations scroll upward, simulating an endless adventure into the ocean. The deeper you go,
the faster everything moves and the darker the water becomes.

## Why did I build it?

I wanted to create something simple. This is my first project so its a bit too simple. I also wanted to practice JavaScript game development with clean, modular code.

## How to test it?

To test it simply open the demo link in the github repository, or locally download it.

## Inspiration

I built this project because I love the ocean, and the diving aspect of it.

## Theme

The theme of this project is **Endless**. This project belongs to the endless theme because of the infinite ocean aspect in this game.
